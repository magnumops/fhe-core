# Logos FHE Accelerator v5.0 (Dual Core Edition)

## Достижения
1.  **Два Ядра:** Реализованы два независимых вычислительных модуля NTT.
2.  **Арбитраж:** Честный доступ к общей памяти (1TB) для ядер и DMA.
3.  **Безопасность:** Внедрен Hazard Unit для предотвращения конфликтов.
4.  **Софт:** Асинхронный планировщик задач на Python.

## Статус
Проект успешно прошел все тесты Фазы 5, включая стресс-тесты и параллельное исполнение.
Архитектура готова к синтезу на FPGA (с заменой Mock-ядер на реальные).
