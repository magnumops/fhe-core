# Предложение для Кодекса Вайбкодера v1.9

### Правило №3.7: Протокол "Криминалистической Таблицы" (The Forensic Table Protocol)
* **Триггер:** Возникновение любой ошибки (Build, Runtime, Test Failure) в ходе выполнения задачи.
* **Действие:** AI-ассистент обязан вести накопительную таблицу "Неудачных Попыток" (Forensic Table) **только для текущего рабочего дня (текущей задачи)**.
* **Формат:** Таблица должна включать колонки: ID попытки, Гипотеза/Действие, Симптом, Root Cause.
* **Обязательство:** При каждой следующей попытке (итерации) в рамках того же дня, AI-ассистент **обязан** показывать эту таблицу целиком, добавляя в нее новую запись.
* **Запрет:** Категорически запрещено скрывать или обнулять историю ошибок текущего дня до момента успешного создания Чекпойнта.
