# Предложение для Кодекса Вайбкодера v1.10
## Раздел 4: Ускоренные Команды (Shortcuts)

### Правило №4.11: Протокол Ускоренных Подтверждений
Для минимизации задержек вводятся следующие семантические сокращения:
* **OK / ok** — "Принято, выполняй следующий логический шаг плана". (Эквивалент: "Дальше").
* **ЧКП / CKP** — "Создай Checkpoint". Ассистент обязан выполнить `git add .` и `git commit`.
* **ГП / GP** — "Выполни Git Push". Ассистент обязан выполнить `git push origin HEAD`.
* **ЧКП ГП** — Комбинированная команда: Коммит + Пуш.

При получении этих команд Ассистент выполняет действие без лишних рассуждений.
